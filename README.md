## Project info

Marketing site and landing page for PaySimply, a tool for sharing UPI payment links and QR codes without a traditional payment gateway.

## Local development

Requirements:

- Node.js and npm installed (recommend using nvm)

Install and run:

```sh
npm install
npm run dev
```

Then open the printed `http://localhost:...` URL in your browser.

## Tech stack

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Build and preview

```sh
npm run build
npm run preview
```
